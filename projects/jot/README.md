# jot

`esp01` with a M5 cardkbd over I2C.

| esp01    | cardkbd (GROVE) |
|----------|-----------------|
| VCC 3.3V | 3.3V (Red)    3 |
| GND      | GND  (Black)  4 | 
| 0        | SCL  (Yellow) 1 |
| 2        | SDA  (White)  2 |
